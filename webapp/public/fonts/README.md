# فونت‌ها

سایت با این ترتیب فونت لود می‌کند:

1. **IRANSansX** — از همین پوشه (لوکال)
2. **Dana** — از CDN فونت‌ایران
3. **Vazirmatn** — از jsDelivr (رایگان، همیشه در دسترس)

برای فعال‌شدن ایران‌سنس، فایل‌های زیر را در همین پوشه قرار دهید:

```
IRANSansX-Regular.woff2
IRANSansX-Medium.woff2
IRANSansX-DemiBold.woff2
IRANSansX-Bold.woff2
IRANSansX-Black.woff2
```

> ایران‌سنس و دانا فونت‌های تجاری‌اند. لایسنس را از [fontiran.com](https://fontiran.com) تهیه کنید.
> تا وقتی فایل‌ها را نگذاشته‌اید، سایت خودکار روی Dana و بعد Vazirmatn می‌افتد و ظاهرش سالم می‌ماند.
